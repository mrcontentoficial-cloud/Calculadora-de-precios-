# Calculadora de Precios · Mr Content

Herramienta interna de cotización para **Mr Content · Agencia Creativa**.

## Qué hace
- Calcula tarifas armando planes por servicio (reels, posts, carruseles, historias, video, fotografía, copy, sitios web).
- Recargos estratégicos (urgencia, exclusividad, estrategia, gestión de pauta) y descuento por permanencia.
- Piso mínimo configurable que avisa cuando una cotización queda por debajo del valor del trabajo.
- Guarda planes en el navegador.
- Genera una **cotización en PDF** con la identidad de marca (logo, tipografías, colores) lista para compartir al cliente.
- Captura datos del cliente, ejecutivo y fecha, con vigencia automática de 15 días hábiles.

## Uso
Abre `index.html` (o `calculadora-mr-content.html`) en cualquier navegador. No requiere instalación.

Para uso 100% sin conexión, la librería de PDF se carga desde internet; sin conexión, el botón abre el diálogo de impresión del navegador ("Guardar como PDF").

## Online
Si se activa GitHub Pages en este repositorio, la calculadora queda accesible desde el navegador sin descargar nada.
